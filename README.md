# Soul Guardian Prototype

A Roguelike action game built with Python and Pygame Community Edition (`pygame-ce`).

## Features
- **Dungeon Generation**: Randomly generated rooms and corridors.
- **Level Progression**: Find the **Purple Portal** to advance to the next level.
- **Difficulty Scaling**: Enemies spawn faster and have more HP as you progress.
- **Weapon System**: Switch between Pistol, Shotgun, and MachineGun (Press `Q`).
- **Combat**: Aim with mouse, Left Click to shoot.
- **Enemies**: Chaser enemies that spawn and navigate towards the player.
- **Game States**: Start Menu, Gameplay, and Game Over screens.
- **Camera**: Follows the player through the dungeon.

## Installation

1.  Ensure you have Python installed.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## How to Run

Run the game using:
```bash
python main.py
```

## Controls
- **W, A, S, D**: Move
- **Mouse**: Aim
- **Left Click**: Shoot
- **Q**: Switch Weapon
- **Enter**: Start Game (in Menu)
- **R**: Restart Game (in Menu or Game Over)
- **Esc**: Pause / Menu
